# Neon Divide

Modern arcade game inspired by Qix/Erix with Cyberpunk/VHS aesthetics.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Database:** NeonDB (PostgreSQL)
- **Auth:** Clerk
- **ORM:** Prisma

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Game Mechanics

Control a neon cursor to capture territory while avoiding enemies. Features roguelike progression with upgrades between levels.
